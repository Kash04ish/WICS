# 🌸 ShaktiAI — Career & Financial Independence for Women

> An AI-powered companion that empowers women across India to achieve career growth and financial independence — regardless of age, background, or experience.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Powered by Claude](https://img.shields.io/badge/Powered%20by-Claude%20AI-orange)](https://anthropic.com)

## 🎯 Problem Statement

Despite making up 48% of India's population, women face:
- **Gender Pay Gap**: Women earn 19% less than men for equivalent work (ILO 2023)
- **Low Financial Literacy**: 80% of Indian women have never made an independent financial decision
- **Career Interruptions**: 48% leave workforce after marriage/childbirth with no re-entry path
- **Rights Unawareness**: Most women unaware of POSH Act, maternity leave, Equal Pay Act
- **Access Barriers**: Quality career/financial guidance is expensive and inaccessible

## 💡 Solution: ShaktiAI

An AI-powered chatbot that provides free, personalized, judgment-free career and financial guidance.

| Module | Description |
|--------|------------|
| 💼 Career | Salary negotiation, job search, promotions, career pivots |
| 💰 Finance | SIP/PPF investing, budgeting, emergency funds |
| ⚖️ Rights | POSH Act, maternity leave, Equal Pay Act |
| 📚 Skills | Free courses, freelancing, entrepreneurship |
| 🏛️ Govt Schemes | Mudra Yojana, Mahila Udyam Nidhi, Stand Up India |

## 🚀 Quick Start

```bash
# Option 1: Open directly
open index.html

# Option 2: Local server
python3 -m http.server 3000
```

## 🏗️ Tech Stack
- Frontend: Vanilla HTML/CSS/JS (zero dependencies)
- AI: Claude Sonnet via Anthropic API
- Deployment: GitHub Pages / Netlify / Vercel (static)

## 🌍 Impact & Scalability
- Zero cost to users
- Expandable to 20+ Indian languages
- WhatsApp bot deployment ready
- NGO white-labeling ready
